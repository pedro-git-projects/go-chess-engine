# Go Chess


## What is it 

Go chess is a chess engine written in Go.

## Features

- [x] Calculate and execute only legal piece movements 
	- [x] Pawn 
	- [x] Rook 
	- [x] Knight 
	- [x] Bishop 
	- [x] Queen 
	- [x] King 
- [ ] Calculate when the king is in check 
- [ ] Accept castles 
- [ ] Implement pawn promotion
- [x] Handle turns 
- [x] Expose API via JSON marshalling
